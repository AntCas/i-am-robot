Server Verify Example

Customer backends should POST the result token to /api/verify with their secret key before gating content. 

curl -X POST "$API_BASE/api/verify" \
  -H "Content-Type: application/json" \
  -d '{
    "secret": "secret_demo_abc",
    "resultToken": "..."
  }'
